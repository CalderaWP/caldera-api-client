    caldera-api-client

    JavaScript API client for Caldera Forms and Caldera Forms Pro

    Forked from: https://www.npmjs.com/package/npm-module-es6-boilerplate

    🌋 Documentation

    Included

    • babel - Transpile ES6
    • Jest - Tests, includes coverage report
    • Coveralls - Ensure that all your new code is fully covered, and see coverage trends emerge.
    • Travis - Deploy and test.
    • Flow - Type Checking
    • esDoc - Documentation

    • esLint - Enforces tabs over spaces and other lints.

    • Auto publish to npm on every build.

    Testing

    This boilerplate uses Facebook Jest. Test are in the directory named __tests__ at any level and start writing tests.

    • Run test watcher
      yarn test

    Scripts

    Build for release

    • yarn build
    • Runs tests, and flow type checks and Babel compiles with minification and source map.

    This script calls a pre, compile, and post subcommands. For consistency, please call other scripts at those three events.

    Tests

    • yarn test : Test Watcher
    • npm test:once : Run Tests once
